# C# — Dominando a Orientação a Objetos

Repositório com exercícios e projeto final desenvolvidos durante o curso **C#: Dominando a Orientação a Objetos** da [Alura](https://www.alura.com.br), ministrado pelos instrutores [Guilherme Lima](https://linktr.ee/guilimadev) e [Daniel Portugal](https://cursos.alura.com.br/user/daniel-portugal).

---

## 📚 Sobre o curso

Curso de 8 horas aprofundando Orientação a Objetos com C#, cobrindo namespaces, herança, interfaces, métodos virtuais, encapsulamento com IEnumerable e integração com IA (Google Gemini). Inclui exercícios práticos e entrega de projeto final.

## 🧠 Conceitos praticados

- Namespaces e organização de projetos em pastas
- Herança (`abstract`, `virtual`, `override`, `base`)
- Interfaces (`interface`, implementação múltipla)
- Encapsulamento com `IEnumerable<T>` vs `List<T>`
- Alias de `using` para resolver conflito de nomes
- Métodos e classes `static`
- Composição entre classes
- Integração com IA via Google Gemini API
- Expression-bodied members (`=>`)
- LINQ (`.Sum()`, `.Average()`, `.OrderBy()`)
